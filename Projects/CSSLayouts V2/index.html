Learning CSS Layouts

Box Models:
  • Width
  • Height
  • Padding
  • Margain

Width Box Model

The width CSS property speci es the width of an element's content area, and it looks like this: width: 10px; Alternatively, it could also look like this: width: 10px 50px;

Height Box Model

The height CSS property speci es the height of an element's content area, and it looks like this: height: 200px;

Padding Box Model

The padding CSS property creates a space on all four sides of an element on the inside, between its content and the edges, and it looks like this: padding: 10px; Alternatively, it could also look like this: padding: 10px 15px;

Margain Box Model

The margin CSS property de nes an outside space on one, two, three or all four sides of an element, and it looks like this: margin: 10px; Alternatively, it could also look like this: margin: 10px 15px;
